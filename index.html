发表于 2019-06-06

Shiro

Apache Shiro是一个强大易用的Java安全框架，提供了认证、授权、加密和会话管理等功能，对于任何一个应用程序，Shiro都可以提供全面的安全管理服务。相比其他安全框架，Shiro要简单很多。

简介

Shiro提供的功能点包括：

Authentication：身份认证/登录，验证用户是不是拥有相应的身份。

Authorization：授权，即权限验证，验证某个已认证的用户是否拥有某个权限。

Session Manager：会话管理，即用户登录后就是一次会话，在没有退出前，它的信息都在会话中。

Cryptography：加密，保护数据的安全性。

Caching：缓存，比如用户登录后，其用户信息、拥有的角色/权限不必每次去查。

Web Support：Web支持，可以非常容易地集成到Web环境。

Concurrency：支持多线程应用的并发验证。

Testing：提供测试支持。

Run As：允许一个用户假装为另一个用户的身份进行访问。

Remember Me：记住我，一次登录后，下次再来不用再登录。

关键概念

Subject: the word subject is a security term that basically means “the currently executing user”.

SecurityManager: the subject’s ‘behind the scenes’ counterpart is the SecurityManager.

jQuery学习 - AJAX

发表于 2019-06-04

jQuery - AJAX

AJAX是与服务器交换数据的技术，在不重载全部页面的情况下实现对部分网页的更新。

jQuery-AJAX load()

load()方法从服务器加载数据，并把返回的数据放入被选元素中。

语法：

$(selector).load(url, data, callback);

（必需）url: 希望加载的url。

（可选）data: 与请求一起发送的查询字符串键值对集合。

（可选）callback: load()方法完成后所执行的函数名称。

jQuery-AJAX get()和post()

两种在客户端和服务端进行请求-响应的常用方法是：GET和POST。

  • GET - 从指定的资源请求数据。
  • POST - 向指定的资源提交要处理的数据。

jQuery $.get()

$.get()方法通过HTTP GET请求从服务器上请求数据。

语法：

$.get(url, callback);

jQuery $.post()

$.post()方法通过HTTP POST请求向服务器提交数据。

语法：

$.post(url, data, callback);

  • html() 设置或返回所选元素的内容（包括HTML标记）

jQuery添加元素

  • empty() - 从被选元素中删除子元素

什么是jQuery？

jQuery是一个JavaScript函数库。

jQuery是一个轻量级的“写得少，做的多”的JavaScript库。

jQuery库包含以下功能：

  • HTML 元素选取
  • HTML 元素操作
  • CSS 操作
  • HTML 事件函数
  • JavaScript特效和动画
  • HTML DOM 遍历和修改
  • AJAX
  • Utilities

jQuery版本问题

1.X：兼容IE6/7/8，使用最为广泛，官方只做BUG维护，功能不再新增。一般项目，使用1.X版本就足够。

2.X：不兼容IE6/7/8，很少有很用。

3.X：不兼容IE6/7/8，只支持最新的浏览器，很多老的jQuery插件不支持这个版本。

jQuery语法

通过jQuery，可以选取（查询，query）HTML元素，并对它们执行“操作”（actions）。

基础语法：$(selector).action()

  • 美元符号（$）定义jQuery

  • 选择符（selector）查询和查找HTML元素

  • action()执行对元素的操作
