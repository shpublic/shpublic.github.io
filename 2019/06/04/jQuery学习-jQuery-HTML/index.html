jQuery学习-jQuery HTML

发表于 2019-06-04

jQuery 获取内容和属性

获得内容 - text()、html()以及val()

  • text() 设置或返回所选元素的文本内容
  • html() 设置或返回所选元素的内容（包括HTML标记）
  • val() 设置或返回表单字段的值

获取属性 - attr()

  • attr() 获取属性值

jQuery 设置内容和属性

设置内容 - text()、html()以及val()

  • text() 设置或返回所选元素的文本内容
  • html() 设置或返回所选元素的内容（包括HTML标记）
  • val() 设置或返回表单字段的值

text()、html()以及val()的回调函数

回调函数有两个参数：被选元素列表何种当前元素的下标，以及原始值。

设置属性 - attr()

jQuery attr()方法也用于设置/改变属性值。

jQuery添加元素

添加新的HTML内容：

  • append() - 在被选元素的结尾插入（仍在该元素内）
  • prepend() - 在被选元素的开头插入
  • after() - 在被选元素之后插入（在该元素外）
  • before() - 在被选元素之前插入

jQuery删除元素

  • remove() - 删除被选元素以及子元素
  • empty() - 从被选元素中删除子元素
