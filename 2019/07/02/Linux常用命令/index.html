Linux文件基本属性

Linux系统对不同的用户访问同一文件（包括目录文件）的权限做了不同的规定。可以使用ll或者ls -l命令来显示一个文件的属性以及文件所属的用户和组。

drwxr-xr-x是该文件（目录）的属性，第一个字符代表了文件类型。“d”代表这是一个目录文件。

还有其他类型的文件：

  • d 目录
  • - 文件
  • l 链接文档(link file)
  • b 表示为装置文件里面的可供储存的接口设备（可随机存取装置）
  • c 表示为装置文件里面的串行端口设备，例如键盘、鼠标（一次性读取装置）

接下来的字符中，以三个为一组，且均为rwx的三个参数的组合，其中r代表可读，w代表可写，x代表可执行。注意这三个权限的位置不会变，如果没有权限，则出现减号-。

Linux文件属主和属组

对于文件来说，它都有一个特定的所有者，也就是对该文件具有所有权的用户。同时，在Linux系统中，用户是按照组分类的，一个用户属于一个或多个组。文件所有者以外的用户又可以分为文件所有者的同组用户和其他用户。对于root用户来说，一般情况下，文件的权限对其不起作用。

更改文件属性

chgrp 更改文件属组

chgrp [-R] 属组名 文件名

参数选项：

  • -R 递归更改文件属组，就是在更改某个目录文件的属组时，如果加上 -R ，那么该目录下的所有文件的属组都会更改。

chown 更改文件属主，也可以同时更改文件属组

chown [-R] 属主名 文件名

chown [-R] 属主名: 属组名 文件名

chmod 更改文件9个属性

文件设置属性有两种方式，一种是数字，一种是符号。

9个属性，分别是owner/group/others三种身份各自的read/write/execute权限。用数字来表示权限时，使用一个3位的二进制数来表示。当具有某个权限时，对应于1，否则对应于0。比如属主的权限为rwx，拥有所有权，那么对应的数字就是111(二进制)，用十进制来表示就为7；属组的权限为r-x，不具有写权限，那么对应的数字就是101（二进制），用十进制来表示就是5；其他用户的权限也为r-x，那么也为101（5）。那么该文件的属性可以表示为755。

chmod [-R] xyz 文件或目录

xyz为权限对应的数字。

这里原来有一个demo.txt文件，属性rw-r--r--(644)，通过chmod 755 demo.txt修改后，权限变为rwxr-xr-x(755)。

另外，我们也可以通过符号方式来修改权限。规则如下：

chmod [-R] [u/g/o/a] [+/-/=] [r/w/x] 文件或文件夹

  • u/g/o/a表示用户身份，a表示全部身份。
  • +/-/=表示加入/除去/设定某个权限。
  • r/w/x表示权限。

比如上面的例子中，将权限改为原来的644权限。还可以通过+/-来增加或者除去权限。

Linux文件与目录管理

Linux的目录结构为树状结构，最顶级的目录为根目录/。

相对路径和绝对路径

相对路径

路径的写法不是由根目录/写起，例如由 /usr/share/doc 要到 /usr/share/man 底下时，可以写成： cd ../man。

绝对路径

路径的写法由根目录/写起，比如/usr/share/doc这个目录。

处理目录的常用命令

  • ls 列出文件
  • cd 切换目录
  • pwd 显示当前目录
  • mkdir 创建一个新的目录
  • rmdir 删除一个空的目录
  • cp 复制文件或者目录
  • rm 移除文件或者目录
  • mv 移动文件或者目录，或者修改文件、目录的名称

可以使用 man [命令]查看各个命令的使用文档，如man cp。

文件内容查看

使用以下命令来查看文件的内容。

  • cat 由第一行开始显示文件内容。

  • tac 从最后一行开始显示，cat倒着写。

  • nl 显示的时候显示行号。

  • more 一页一页地显示文件内容。
  • less 与more类似，但是可以往前翻页。
  • head 只看头几行。
  • tail 最看最后几行。

Linux yum 命令

yum (Yellow dog Updater, Modified) 是一个在Fedora和RedHat以及SUSE中的软件包管理器。基于RPM包管理，能从指定的服务器自动下载并且安装，可以自动处理依赖关系，并且一次安装所有依赖的软件包，无须繁琐地一次次下载、安装。

语法：

yum [options] [command] [package …]

  • options 可选，选项包括 -h (帮助)，-y （当安装过程提示选择全部为yes），-q（不显示安装的过程）等等。
  • command 要进行的操作。
  • package 操作的对象。

yum常用命令

  • yum check-update 列出所有可更新的软件清单
  • yum update 更新所有软件
  • yum install 安装指定的软件
  • yum update 更新指定的软件
  • yum list 列出所有可安装的软件
  • yum remove 删除指定的软件包
  • yum search 查找软件包
