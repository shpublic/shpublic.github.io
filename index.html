什么是jQuery？

jQuery是一个JavaScript函数库。

jQuery是一个轻量级的“写得少，做的多”的JavaScript库。

jQuery库包含以下功能：

  • HTML 元素选取
  • HTML 元素操作
  • CSS 操作
  • HTML 事件函数
  • JavaScript特效和动画
  • HTML DOM 遍历和修改
  • AJAX
  • Utilities

jQuery版本问题

1.X：兼容IE6/7/8，使用最为广泛，官方只做BUG维护，功能不再新增。一般项目，使用1.X版本就足够。

2.X：不兼容IE6/7/8，很少有很用。

3.X：不兼容IE6/7/8，只支持最新的浏览器，很多老的jQuery插件不支持这个版本。

jQuery语法

通过jQuery，可以选取（查询，query）HTML元素，并对它们执行“操作”（actions）。

基础语法：$(selector).action()

  • 美元符号（$）定义jQuery

  • 选择符（selector）查询和查找HTML元素

  • action()执行对元素的操作

环境配置-Java开发环境配置

发表于 2019-05-29

Java环境配置

这里JDK的安装过程省略，只讲环境变量怎么配，所有环境变量都增加在系统变量中。

新建 $\rightarrow$ 变量名”JAVA_HOME”，变量值”C:\Program Files\Java\jdk1.8.0_201”(JDK安装目录)。

编辑 $\rightarrow$ 变量名”Path”，添加两个新的变量：

​ （1）%JAVA_HOME%\bin

​ （2）%JAVA_HOME%\jre\bin

如果不能单独添加，变量名是写在一行的话，在原来变量名后添加：

“;%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin”

新建 $\rightarrow$ 变量名”CLASSPATH”，变量值为：

“.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar”

要验证是否配置正确，可在命令行下输入javac、java -version等命令，能出现Java相关信息，则说明配置完成了。
