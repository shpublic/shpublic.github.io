hexo入门-Writing

发表于 2019-05-28

hexo 写作

通过以下方式新建一个页面：

$ hexo new [layout]

比如：

$ hexo new post “hexo入门-Writing”

这里post是页面布局方式，可选项，默认为post。

Layout

在Hexo有三种页面布局方式：post，page以及draft。不同布局的页面会被保存在不同的文件夹下，具体如下：

Layout Path
post source/_posts
page source
draft source/_drafts

Filename

默认情况下，Hexo使用title作为文件名，你也可以在_config.yml文件中修改默认配置，比如:year-:month-:day-:title.md。有以下这些占位符可以使用：

Placeholder Description
:title 题目
:year 年份，比如2019
:month 月份（补零），比如05
:i_month 月份（不补零），比如5
:day 天数（补零），不如09
:i_day 天数（不补零），比如9
